=====================================================
  은퇴자금 포트폴리오 AI  — 시작 가이드
=====================================================

[ 폴더 구조 ]
  00_setup.ipynb         ← 처음 한 번만 실행 (라이브러리 설치, DB 초기화)
  01_data_input.ipynb    ← 자산 입력 및 버킷 분류
  config.yaml            ← 개인 설정 (생활비, 목표 비중 등)
  assets_sample.csv      ← 자산 입력 양식 샘플
  .env.template          ← API 키 설정 양식 (복사 후 .env 로 저장)
  시작하기.txt           ← 지금 보고 계신 파일


[ 첫 실행 순서 — 총 30분 ]

  STEP 1. 터미널(PowerShell)에서 아래 명령 실행:
  ─────────────────────────────────────────────
  cd "C:\Users\kwak5\OneDrive\바탕 화면\은퇴포트폴리오AI"
  jupyter lab
  ─────────────────────────────────────────────
  → 브라우저가 자동으로 열립니다

  STEP 2. 브라우저에서 00_setup.ipynb 열기
  → 셀을 순서대로 실행 (Shift+Enter)
  → 라이브러리 설치 (2~5분)
  → .env 파일 생성 후 OpenAI API 키 입력
  → DB 초기화

  STEP 3. assets_sample.csv 를 복사해서 assets.csv 로 저장
  → 메모장이나 엑셀로 열어서 실제 보유 자산으로 수정

  STEP 4. 01_data_input.ipynb 실행
  → 자산 로드 → 버킷 분류 → 차트 확인


[ API 키 설정 방법 ]

  1. .env.template 파일을 복사해서 .env 로 저장
  2. 메모장으로 .env 열기
  3. OPENAI_API_KEY=sk-여기에 실제 키 붙여넣기
  4. 저장 후 닫기

  ※ .env 파일은 절대 다른 사람에게 보내거나 공유하지 마세요.


[ 자산 유형(asset_type) 값 ]
  cash    → 예금, CMA, MMF, 파킹통장
  bond    → 채권, 단기채 ETF, 국고채
  tdf     → TDF (생애주기 펀드)
  equity  → 주식형 ETF, 펀드, 개별주식
  income  → 리츠, 배당 ETF, 인컴 자산
